Hello world! I'm a postdoctoral researcher at the User Interfaces group led by Antti Oulasvirta at Aalto University. My current work aims to combine cognitive modeling and machine learning to improve human-computer interactions. Here are some questions I am currently working on:

  • How to improve the personnalization of self-teaching applications, by making inferences on the user’s cognitive characteristics.
  • How to apply decision-making models to naturalistic human-computer interaction data (i.e. gaming data, browsing data, etc.).
